# Martin Tran's Portfolio

A clean, minimal portfolio website built with Next.js, Tailwind CSS, and Shadcn UI. Features an email contact form and project showcase.


## Features

- Minimal design with Shadcn UI
- Light/dark mode toggle
- Contact form with email integration
- Responsive mobile design
- Project showcase with tags and links

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Shadcn UI
- TypeScript
- Framer Motion (animations)
- Resend (email service)

## Getting Started

```bash
git clone https://github.com/martintranthecoder/martin_tran_portfolio my-portfolio
cd my-portfolio
npm install
cp .env.example .env.local
# add your API keys to .env.local
npm run dev
```
